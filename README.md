To-Do List con Lista Enlazada

Proyecto web desarrollado con Python que implementa una lista de tareas 
usando una lista enlazada simple.


## Funcionalidades

- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Las tareas completadas se muestran con texto tachado

## Tecnologías usadas

- Python 3
- Flask
- HTML y CSS
- Jinja2

## Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Activar el entorno virtual: `source .venv/bin/activate`
3. Ejecutar el servidor: `python3 app.py`
4. Abrir en el navegador: `http://localhost:8000`

## Notas

El entorno virtual `.venv` no está subido a GitHub porque contiene 
las librerías instaladas localmente y pesa mucho. 
Para instalar Flask: `pip install flask`
